Feature : Question Editor

Bạn chỉ cần đưa:

Feature: <tên feature>
Scope: <giới hạn rõ>

Ví dụ:

Feature: Question Editor (MCQ_SINGLE)
Scope: Create only (no edit)

1. Domain → check thiếu gì
2. Hook → nếu cần thì design + viết
3. Mapper → bổ sung nếu thiếu
4. RPC → verify payload / response
5. UI → chỉ bổ sung component nếu thật sự thiếu

✅ Scope của feature = các “use-case thực tế” của user
