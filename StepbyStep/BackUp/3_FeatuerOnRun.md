Scope của Question List cần chốt trước

Question List không chỉ là “render array”, thường sẽ gồm 5 scope:

1 Basic list
load danh sách question
table/card view
pagination

2 Filter
theo category
question_type
difficulty
keyword search

3 Sort
created_at
difficulty
usage count
updated_at

4 Actions
view detail
edit
delete
duplicate

5 Selection state
bulk delete
bulk tag
bulk move category
